Table below contains rules by which transliteration is performed.

Use selector in the right to choose different rules.

The table is sorted by output codepoint value.

Abbreviations:

  • FVS: Free Variation Selector, a series of invisible characters that modify preceding characters in an undefined way
  • MVS: Mongolian Vowel Separator, an invisible character that is supposed to separate ending vowels, but it's not defined how
  • ZWJ: Zero-width joiner, an invisible character that connects neighboring characters
  • ZWNJ: Zero-width non-joiner, an invisible character that disconnects neighboring characters
